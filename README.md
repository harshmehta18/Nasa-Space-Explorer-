## Setup and Installation

cd nasa-space-explorer

- Backend Setup
cd backend
npm install

- Frontend Setup 

cd ../frontend
npm install

## Run Application

- Start Backend 

cd Back-end
node server.js

- Start Frontend 

cd frontend
npm start 

The application will be available at http://localhost:3000
